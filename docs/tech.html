协议

协议头一般有http、https、ftp等这里写一下匹配的方式：

let protocol = /(?:ht|f)tp(?:s)?(?=:\/\/)/
protocol.exec('http://1234')
// ["http", index: 0, input: "http://1234", groups: undefined]

解释：

  • 使用非捕获组/(?:ht|f)/，只对值判断，但是不将结果，也就是(ht|p)看作是一个捕获组；
  • 使用正向零宽断言(?=:\/\/),匹配后面的://，只有有此符号才算匹配。
